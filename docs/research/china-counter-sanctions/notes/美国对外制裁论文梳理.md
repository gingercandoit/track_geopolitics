## 关键机制

### Entity List 的长臂管辖

两个核心规则：

1. **最低限度规则**（de minimis, EAR §734.4）：外国物项中含美国技术的比例决定是否受管辖
   - 0% 档：半导体、加密、先进节点 IC 设备——含任何美国技术即受管辖（适用中国）
   - 10% 档：适用于古巴/朝鲜/伊朗/叙利亚
   - 25% 档：适用于其余国家（含中国的非特定物项）

2. **外国直接产品规则**（FDPR, EAR §734.9(e)）：
   - 直接产品条款：源自美国管制技术的外国生产物项，向脚注实体出口需许可（默认拒绝）
   - 工厂/主要组件条款：外国工厂使用了美国特定技术作为主要生产组件，其产品向脚注实体出口也需许可（希捷案例）

### SDN 的穿透制裁与次级制裁

1. **穿透制裁**（50% 规则）：SDN 实体直接或间接合计持股 50% 以上的企业，自动受同等制裁。三种场景：
   - 单一 SDN 实体 → 50% 持股 → 子公司被穿透
   - 单一 SDN 实体 → 分别持股多家 → 多家再合计持股第三方 ≥50% → 第三方被穿透
   - SDN 实体直接持股 + 通过已穿透企业间接持股 → 合计 ≥50% → 被穿透

2. **次级制裁**：非美国第三方实体若与 SDN 实体有业务往来，也可能面临制裁。2023-12 EO14114 扩展——即使非美元货币结算也适用。

3. **关键发现**：SDN 内中国实体大多因与俄罗斯/伊朗/朝鲜交易被间接连带制裁，与 Entity List（直接针对中国实体的技术封锁）逻辑完全不同。华为系企业不在 SDN 清单内。

### 涉军清单演变

CCMC(2020-06, 44家) → 2021-06 停用 → 同日启动 CMC(76家) + CMIC(67家)。去重后 110 家。34 家从 CCMC 转入 CMC+CMIC，25 家新增交叉制裁。制裁路径从单一的金融限制（禁止投资）扩展为投资限制 + 进口管制 + 2026-06 后禁止国防部与清单企业签约。

## 核心结论

- 2018 贸易战是结构性拐点：之前制裁分散、增速平缓；之后快速扩张、集中于高科技战略企业
- 内地企业在制裁实体中的占比从 2018 年的 31% 升至 2024 年的 52%
- 三类核心清单各有侧重：Entity List 管技术出口、SDN 管金融冻结和交易禁止、涉军清单管投资和进口
- 三类清单的制裁对象重合度低——Entity List 和 SDN 的重叠很少
