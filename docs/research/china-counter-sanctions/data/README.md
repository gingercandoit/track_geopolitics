# 中国反制裁研究  数据目录

> 最后更新：2026-04-14

本目录存放中国反制裁研究项目的数据文件，按中方（CN）和美方（US）分类。

## 双边配对数据

### 美方DSCA对台军售通知.csv

| 字段 | 说明 |
|------|------|
| 日期 | DSCA通知日期（YYYY-MM-DD），即"Date Report Delivered to Congress" |
| 武器/服务 | 军售内容简述（英文） |
| 金额(百万美元) | 估计总金额（百万美元） |
| 来源 | 数据来源标注，含CN（Congressional Notification）编号 |
| 官方URL | 官方全文链接（Federal Register 或 DSCA 官网） |

与 CN/ 中方两个清单（外交部反制裁清单 + 不可靠实体清单）中"对台军售"相关批次做时间配对分析。

### 美方DSCA对台军售事件.csv

| 字段 | 说明 |
|------|------|
| 日期 | 按 DSCA 通知日期聚合后的事件日期（YYYY-MM-DD） |
| 通知数 | 当日独立 CN 通知条数 |
| 总金额(百万美元) | 当日所有通知金额加总 |
| CN编号 | 当日所有 CN 编号，按通知顺序以分号连接 |
| 武器/服务 | 当日所有通知标题，按通知顺序连接 |
| 来源 | 当日所有来源标注，按通知顺序连接 |
| 官方URL | 当日所有官方链接，按通知顺序连接 |

用于“美方动作 → 中方反制”时滞分析。若研究问题是“每条通知触发一次反应”，使用通知级表；若研究问题是“同日军售波次触发一次反应”，使用事件级表。

### 美方DSCA对台军售_承包商映射.csv

| 字段 | 说明 |
|------|------|
| CN编号 | DSCA Congressional Notification 编号 |
| 日期 | DSCA 通知日期（YYYY-MM-DD） |
| 武器系统 | 军售案标题或核心系统 |
| 可能承包商 | 优先记录 DSCA/Federal Register 明示承包商；若未明示，则记录最可能候选 |
| 证据等级 | A=官方明确点名 principal contractor；B=未点名但型号与官方产品页精确对应、可做强映射；C=非强映射候选，含高可信推断以及竞争采购、多供应商、政府库存转移、框架合同等仅能保留候选的案件 |
| 判断依据 | 承包商判断逻辑，注明是否为竞争采购、政府库存转移或型号映射 |
| 来源URL | 对应官方或公司/辅助来源链接 |

用于把 DSCA 对台军售通知与中方“对台军售”制裁对象做公司层面的命中配对；注意该表并不等同于“最终签约主承包商清单”，尤其在政府库存转移、DLA 供应商框架或竞争采购案件中，只能视为候选映射。

### 美方DSCA对台军售_中方反制配对模板.csv

| 字段 | 说明 |
|------|------|
| 美方军售日期 | DSCA 通知日期（YYYY-MM-DD） |
| 军售内容 | 该笔军售的标题或核心内容 |
| 可能承包商 | 候选承包商，来自承包商映射表或补充判断 |
| 美方URL | 对应 DSCA/Federal Register 公告 |
| 中方外交回应日期 | 中方首次公开发声、回应或表态日期 |
| 中方回应URL | 对应上述外交回应、记者会或声明来源链接 |
| 中方对应制裁日期 | 若后续落到外交部/商务部正式制裁名单，则填写正式制裁日期；若仅有口头回应则留空 |
| 中方制裁URL | 对应正式制裁名单、公告或制裁口径来源链接；若无正式制裁则留空 |
| 制裁清单名称 | 例如“外交部反制裁清单”“商务部不可靠实体清单”“外交部口头声明（未见名单制裁）” |
| 制裁内容 | 中方回应或制裁内容摘要 |
| 该军售类型 | 海军装备、导弹与弹药、无人机等分类 |

用于把每一笔对台军售与“中方回应”及“后续是否形成正式制裁”分开记录。这样可以区分“只有外交表态”与“已形成清单制裁”的两种情况，更适合做反应时滞和工具选择分析。若尚未单独核到回应页面，则“中方回应URL”留空，不再混填“中方制裁URL”。

## 数据来源

- **CN/外交部清单**：中国外交部官网发言人记者会/声明，逐条核实URL
- **CN/商务部清单**：中国商务部官网公告，逐条核实URL
- **US/汇总+分清单**：原始 Stata .dta 文件来自用户前期研究，转为 CSV。**9个分清单已全部更新至2026-04**：CSL 更新6个（SDN/EL/UVL/CMIC/MEU/ISN），DoD/DHS 原始数据源更新3个（CMCC/CMC/UFLPA）。汇总表尚未同步
- **DSCA数据**：现含 `45` 条通知级记录和 `27` 条日期级事件。2026-04-14 完成一次口径修正：补入 2024-11-29 两条遗漏通知、拆分 2024-06-05 合并记录，并更正 2025-12-17 两条 CN 编号。
- **DSCA承包商映射**：`美方DSCA对台军售_承包商映射.csv` 现覆盖全部 `45` 条 CN；其中优先采用 DSCA/Federal Register 原文的 principal contractor 信息。当前证据等级采用三级口径：`A` 为官方点名，`B` 为未点名但可强映射，`C` 为非强映射候选，统一覆盖竞争采购、政府库存转移、多供应商和框架合同等情形。
- **DSCA中方反制配对模板**：`美方DSCA对台军售_中方反制配对模板.csv` 现把“中方外交回应日期/URL”与“中方对应制裁日期/URL”分开记录；仅有发声、未形成正式制裁的案件，只填写回应字段。
