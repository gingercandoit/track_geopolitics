# 特斯拉计划采购中国光伏设备？企业回应：确有此事

**来源**：新京报（东方财富转载）
**日期**：2026-03-20
**URL**：https://finance.eastmoney.com/a/202603203678598148.html

---

特斯拉"扫货"中国光伏的传闻又有后续。3月20日，一则市场传闻称，特斯拉团队计划采购较大规模的中国光伏设备，涉及数家上市公司。

3月20日上午，一家光伏企业向贝壳财经记者证实了上述消息，并透露合同规模在吉瓦级。

今年2月初，特斯拉CEO埃隆·马斯克考察多家中国光伏企业的消息曾数度扰动市场。有设备企业向贝壳财经记者证实，马斯克的考察团近期确实来公司进行了供应链考察。

协鑫集团方面在2月4日晚间对外透露，马斯克旗下公司团队考察了协鑫集团，并表达了在颗粒硅与钙钛矿方面的合作意向。

不过2月的数轮市场情绪中，关键词是"太空光伏"。这是一项被业内认为落地仍需要较长时间的、配套商业航天的光伏应用场景。中国光伏行业协会执行秘书长刘译阳向媒体表示，目前，太空光伏技术仍处于探索和验证的初期阶段，确定明确的技术方向为时尚早。

有部分企业也曾因"蹭热点"遭到监管处罚。

2月28日，双良节能因涉嫌信息披露误导性陈述等违法违规行为，被证监会立案。半个月后，双良节能控股股东双良集团有限公司也因同样事由被立案。

此前于2月12日，双良节能通过微信公众号发布消息称，公司近日先后获得3个海外订单，共计12台高效换热器设备，将用于SpaceX星舰发射基地扩建配套的燃料生产系统。该消息发布后，双良节能股价应声上涨。

上交所出具的监管警示函则指出，当前"商业航天"属于市场较为关注的热点概念，为投资者高度关注。双良节能在微信公众号中发布相关信息时，未说明订单的供货方式、销售规模及对公司整体经营影响较小等具体情况，也未就后续订单的不确定性等情况充分提示风险，"可能对投资者决策产生误导"。
