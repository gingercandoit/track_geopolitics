# 写作风格规范

> 跨所有 Prompt 适用。每次生成文字内容前，必须过一遍这份清单。

---

## 一、禁用词与套话（不得出现）

以下词汇和句式是机器文风的标志，**一律禁止**：

| 禁止 | 原因 |
|------|------|
| "结构性演变/侵蚀/调整/转变" | 万能词，等于没说 |
| "A+B+C 三条主线" / "可提炼为N条主线" | 框架化而非叙述 |
| "呈现出…主旋律" | 假装总结，实为空话 |
| "与此同时" × 3+ | 可以用一次，不可滥用 |
| "标志着…时代的到来" | 新闻通稿常见套话 |
| "值得关注的是" | 废话引导语 |
| "不可逆转地" | 过度断言 |
| "深层矛盾…在极端场景下被迫暴露" | 过度修饰 |
| "折射出…深层不信任" | 类比替代分析 |
| "这一趋势正在…" | 模糊时态 |
| "进一步" × 3+ | 蹭进展词 |

## 二、写作准则

### 白描优先
- **具体事实先于框架**：先说"财政部临时豁免1.4亿桶伊朗原油"，再说意义；不要先说"制裁工具正面临新挑战"再举例
- **数字胜过形容词**：写"76个经济体"而非"史无前例的广泛范围"
- **动词驱动**：用"USTR发起调查""北京限制出境""BIS撤回草案"，不用"政策工具的调整"

### 叙述语气参考
- **南方周末**：白描细节，事件本身说话；少判断，多呈现
- **The Economist**：一句话点破关键矛盾；用个具体例子锚住抽象论点
- **Reuters/NYT**：倒金字塔，最重要的事放第一句；不为读者"总结意义"

### 字数纪律
- **主页摘要卡片（digest_zh）**：80-120汉字，不要超
- **月度总览（overview）**：可以长，但每段有一个核心信息点；每段第一句必须是那个核心点本身，不是引导语
- **事件bullet**：每条一个独立信息点，不叠加

### 段落结构
- 段落首句 = 核心断言（可以直接引用数字/事件）
- 几个支撑细节
- 可以有一句意义，但要具体（"这是4年来首次突破100美元"）而非空泛（"影响深远"）

## 三、典型改写示例

**原文（禁止）**：
> 2026年3月，制裁与经济管制领域呈现三条交织主线：美国OFAC制裁机器的高频运转、欧盟对俄多层制裁的持续推进，以及人民币/CIPS体系对美元制裁基础设施的结构性侵蚀。

**改写（目标）**：
> 3月，美国财政部临时豁免约1.4亿桶伊朗原油的制裁——同日解除30艘俄罗斯油轮限制，以平抑破百的国际油价。OFAC全月在反恐、朝鲜、真主党多线出手；北京则同步将稀土、镓纳入自身的出口管制清单。制裁的工具箱，已是双向的了。

## 四、语言禁忌 × digest_zh 专项

主页摘要卡片（80-120字）的额外要求：
- **绝不照搬月度总览第一句**
- 选取**最具冲击力的1-2个具体事件**，而非主题框架
- 读者读完应有"啊，这个月发生了这件事"的感觉，而非"啊，这个月有三条主线"
- 不用引号"" 包裹框架词（如"规则重建"、"配给制"——除非确实是该词语首次提出的关键概念，且马上有具体解释）
